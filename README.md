## Run Locally

Clone the project

```bash
  git clone https://github.com/Fanndev/E-Commerce_Kelompok3.git
```

Go to the project directory

```bash
  cd E-Commerce_Kelompok3
```

Install dependencies

```bash
  Flutter pub get
```

Start the server

```bash
  flutter run
```
