## Getting Started

Projek kelompok berbasis flutter Tema Ecommerce

Run Locally
Clone the project

  git clone https://github.com/Fanndev/E-Commerce_Kelompok3.git
Go to the project directory

  cd E-Commerce_Kelompok3
Install dependencies

  flutter pub get
Start the server

  flutter run
